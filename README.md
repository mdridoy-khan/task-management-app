# Task Management App

This is a React + Tailwind CSS implementation of a task management interface based on the provided low-fidelity wireframe.

## Setup

1. Clone the repo.
2. Run npm install.
3. Run npm run dev to start the development server.
4. Open http://localhost:5173.

## Deployment

Deploy in vercel

## Pages

- / : Kanban Board (with static drawer)
- /list : List View
